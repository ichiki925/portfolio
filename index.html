WORKS

Vanilla's Kitchen（レシピアプリ・デモ）

準備中

管理者が投稿するレシピを、会員ユーザーが閲覧・お気に入り・コメントできるアプリです。
Firebase認証とストレージ、Laravel API、Nuxt 3で構成し、PC/スマホに最適化しています。

ユーザー機能

  • 会員登録 / ログイン（Firebase Auth）
  • レシピ閲覧・検索（キーワード）
  • いいね / お気に入り一覧に自動反映
  • レシピ詳細からコメント投稿
  • プロフィール編集
  • レスポンシブ対応

管理者機能

  • レシピの投稿・編集・削除
  • コメント一覧・削除（モデレーション）
  • ユーザー数カウント表示
  • 最近更新・最近削除の可視化

技術スタック

  • JavaScript (ES2020+)
  • Vue 3 / Nuxt 3
  • PHP 8.1 / Laravel 8.75
  • Firebase Auth / Firebase Storage
  • MySQL
  • Docker

勤怠アプリ

準備中

スタッフの出退勤管理と修正申請・承認フローを実装した勤怠管理アプリです。
打刻機能、休憩記録、CSV出力機能を備え、管理者による承認フローを実現しています。

ユーザー機能

  • 出勤・退勤打刻（リアルタイム）
  • 休憩開始・終了記録
  • 勤怠履歴の閲覧（月次カレンダー）
  • 勤怠詳細表示
  • 勤怠データの修正申請
  • 申請一覧の確認（承認待ち/承認済み）

管理者機能

  • スタッフ一覧表示
  • 日次勤怠一覧の確認
  • スタッフ別月次勤怠の確認
  • 勤怠データの直接修正
  • 修正申請一覧の確認（承認待ち/承認済み）
  • 修正申請の承認・却下
  • 月別勤怠データのCSV出力

技術スタック

  • PHP 8.2 / Laravel 8.75
  • MySQL 8.0
  • Nginx
  • Docker / Docker Compose
  • Laravel Fortify (認証)

SHARE(Twitter風SNSアプリ)

準備中

投稿・いいね・コメント機能を実装したTwitter風SNSアプリです。
Firebase認証とリアルタイム更新で、ユーザー同士のコミュニケーションを実現しています。

ユーザー機能

  • 会員登録・ログイン（Firebase Authentication）
  • 投稿の作成・表示・削除
  • タイムライン（投稿一覧表示）
  • ユーザー別投稿一覧
  • いいね機能（追加・削除）
  • いいね状態の確認
  • コメント機能（投稿・一覧表示）
  • 投稿詳細表示

  • デモ専用環境設定

  • Firebase 12.2 (Authentication)

  • Nginx 1.21
  • Docker / Docker Compose
